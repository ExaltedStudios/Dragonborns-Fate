Gameplay

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select All Mods -> Create Separator
  2. Name the separator Gameplay

It is especially important to read the descriptions of these mods (you should still read the description of every mod), so you know what has changed and how to use the new features.

Mysticism - A Magic Overhaul

Installation instructions:

  • Main File - Mysticism - A Magic Overhaul 1.2.1

- Heavily balances all vanilla spells and spell progression, as well as adding in new vanilla-friendly spells

Adamant - A Perk Overhaul

Installation instructions:

  • Main File - Adamant - A Perk Overhaul 5.1.8
  • Optional File - Adamant Addons - Shrines and Amulets 5.0.3

- Balanced and streamlined perk overhaul that aims to provide compelling choices and smooth progression

Blade and Blunt - A Combat Overhaul

Installation instructions:

  • Main File - Blade and Blunt - A Combat Overhaul 1.3.1

- Combat overhaul focused on resource management, also includes better enemy AI, tweaked damage multipliers, and more

Apothecary - An Alchemy Overhaul

Installation instructions:

  • Main File - Apothecary - An Alchemy Overhaul 1.1.4
  • Apothecary - Food and Drink Addon 1.0.2

- Complete overhaul and rebalancing of the game's alchemy system

Mundus - A Standing Stone Overhaul

Installation instructions:

  • Main File -Mundus - A Standing Stone Overhaul 1.6.2

- Improves standing stone effects to create a meaningful choice between them

Aetherius - A Race Overhaul

Installation instructions:

  • Main File - Aetherius - A Race Overhaul 2.5.2

- Overhauls the game's racial bonuses to make each race feel meaningul and unique

Manbeast - A Werewolf Overhaul

Installation instructions:

  • Main File - Manbeast - A Werewolf Overhaul 1.2.0

- Lightweight werebeast overhaul that makes them a viable choice without making them overpowered

Scion - A Vampire Overhaul

Installation instructions:

  • Main File - Scion - A Vampire Overhaul 1.2.2

- Lightweight vampire overhaul that makes them a viable choice without making them overpowered

SunHelm Survival and Needs

Installation instructions:

  1. Download the Main File - SunHelm Survival 3.0.1 through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should just select the Diseases option on the second page
I personally also select the Disable Cold option on the second page of the FOMOD as I think the system is much more of an annoyance than anything else, but this is completely optional.
  1. Finish the installation normally

- Adds a fully configurable needs system as well as other hardcore features

Just like any other mod in this guide, this can be skipped without any problems if you do not want it.

Encounter Zones Unlocked

Installation instructions:

  • Main File - Encounter Zones Unlocked SE 1.4

- Prevents the game from locking an area's encounter zone level when you first enter it

Economy Overhaul

Installation instructions:

  1. Download the Main File - Economy Overhaul and Speechcraft Improvements 3.0 Fomod Installer through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should select the Lite Version option on the first page and the Quests Are In Skyrim Patch option on the second page
  3. Finish the installation normally

- Balances the economy by changing buy/sell prices, adjusting certain quest rewards, and other miscellaneous changes

Scarcity

Installation instructions:

  1. Download the Scarcity SE - Less Loot Mod 1.1 through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should only select Install
  3. Finish the installation normally

- Globally reduces all types of loot, making the economy much more balanced

Open World Loot - Encounter Zone and Loot Overhaul

Installation instructions:

  • Main File - OWL Main File 1.3.0
  • Update File - OWL Randomized Special Loot 1.2.5
  • Miscellaneous File - Patch - Scarcity 1.2.6

- Overhauls all the game's encounter zones and loot tables to allow for more freedom to explore the open world

Open World Loot - Vanilla Progression

Installation instructions:

  • Main File - OWL - Vanilla Progression 1.2
  • Optional File - OWL - OWL - Vanilla Progression - Randomized Special Loot 1.0

- Makes Open World Loot use vanilla's weapon progression tiers

Disarmless

Installation instructions:

  • Main File - Disarmless Draugrs 0.1

- Removes the incredibly frustrating disarm spell from Draugrs
