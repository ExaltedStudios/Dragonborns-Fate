Gameplay

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator.
  2. Name the separator Gameplay.

It is especially important to read the descriptions of these mods (you should still read the description of every mod), so you know what has changed and how to use the new features.

Aetherius - A Race Overhaul

Installation instructions:

  • Main File - Aetherius - A Race Overhaul

Complete overhaul of human, elf, and beast races designed to balance the races and add powerful new gameplay options.

Mysticism - A Magic Overhaul

Installation instructions:

  • Main File - Mysticism - A Magic Overhaul

Complete overhaul of the magic system, designed to balance existing spells and add powerful new magical effects.

Adamant - A Perk Overhaul

Installation instructions:

  • Main File - Adamant - A Perk Overhaul

Complete overhaul of the perk trees designed to balance existing skills and add powerful new perks.

Mundus - A Standing Stone Overhaul

Installation instructions:

  • Main File - Mundus - A Standing Stone Overhaul
  • Optional File - Mundus - USSEP Tower Stone Patch

Complete overhaul of the Standing Stones designed to balance the stone bonuses and add powerful new gameplay options.

Arena - An Encounter Zone Overhaul

Apothecary - An Alchemy Overhaul

Installation instructions:

  • Main File - Apothecary - An Alchemy Overhaul
  • Main File - Apothecary - Food and Drink Addon
    • (When prompted, expand the names drop-down and select the second one)
  • Miscellaneous File - Apothecary - Fishing Patch
    • (When prompted, select the name that matches the file on nexus)
  • Miscellaneous File - Apothecary - Food and Drink Addon - Survival Mode Patch
    • (When prompted, select the name that matches the file on nexus)
  • Miscellaneous File - Apothecary - Food and Drink Addon - USSEP Patch
    • (When prompted, select the name that matches the file on nexus)
  • Miscellaneous File - Apothecary - Rare Curios Patch
    • (When prompted, select the name that matches the file on nexus)
  • Miscellaneous File - Apothecary - Saints and Seducers Patch
    • (When prompted, select the name that matches the file on nexus)

Complete overhaul of the alchemy, food, and alcohol systems designed to balance existing options while adding powerful new effects.

Pilgrim - A Religion Overhaul

Installation instructions:

  • Main File - Pilgrim - A Religion Overhaul
  • You will be prompted with a FOMOD installer, click Install

Complete overhaul of the religion system that adds dozens of new deities and with powerful shrine effects.

Thaumaturgy - An Enchanting Overhaul

Hand to Hand - An Adamant Addon

Installation instructions:

  • Main File - Hand to Hand - An Adamant Addon

Addon for Adamant - A Perk Overhaul that merges Lockpicking and Pickpocketing into a single skill and adds a new perk tree for Hand to Hand combat.

Sensible Bribes

Installation instructions:

  • Main File - Sensible Bribes

Dynamic Mercenary Fees

Installation instructions:

  • Main File - Dynamic Mercenary Fees

Makes mercenaries ask for more or less money for their services depending on their level, your and their race and how good you are at bartering.
