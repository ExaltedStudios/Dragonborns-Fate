Gameplay

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Gameplay.
It is especially important to read the descriptions of these mods (you should still read the description of every mod), so you know what has changed and how to use the new features.

Aetherius - A Race Overhaul

Installation instructions:

  • Main File - Aetherius - A Race Overhaul
Complete overhaul of human, elf, and beast races designed to balance the races and add powerful new gameplay options.

Mysticism - A Magic Overhaul

Installation instructions:

  • Main File - Mysticism - A Magic Overhaul
  • Optional Files - Mysticism - Jump Spell Addon (When prompted, select the name that matches the file on Nexus)
  • Optional Files - Mysticism - Survival Spells Addon (When prompted, select the name that matches the file on Nexus)
  • Optional Files - Mysticism - Jump and Survival Spells (Tiny) Leveled List Patch (When prompted, select the name that matches the file on Nexus)
Complete overhaul of the magic system, designed to balance existing spells and add powerful new magical effects.

Adamant - A Perk Overhaul

Installation instructions:

  • Main File - Adamant - A Perk Overhaul
  • Optional Files - Adamant - Smithing Addon (When prompted, select the name that matches the file on Nexus)
Complete overhaul of the perk trees designed to balance existing skills and add powerful new perks.

Mundus - A Standing Stone Overhaul

Installation instructions:

  • Main File - Mundus - A Standing Stone Overhaul
  • Optional File - Mundus - USSEP Tower Stone Patch (When prompted, select the name that matches the file on Nexus)
Complete overhaul of the Standing Stones designed to balance the stone bonuses and add powerful new gameplay options.

Arena - An Encounter Zone Overhaul

Installation instructions:

  • Main File - Arena - An Encounter Zone Overhaul
Complete overhaul of the encounter zones designed to increase difficulty and provide an organic sense of progression.

Blade and Blunt - A Combat Overhaul

Installation instructions:

  • Main File - Blade and Blunt - A Combat Overhaul
Complete overhaul of the combat designed to increase difficulty and encourage active resource management.

Scion - A Vampire Overhaul

Installation instructions:

  • Main File - Scion - A Vampire Overhaul
  • Also get the USSEP patch here
Complete overhaul of the Vampire system designed to balance existing mechanics and add powerful new abilities.

Manbeast - A Werewolf Overhaul

Installation instructions:

  • Main File - Manbeast - A Werewolf Overhaul
  • Also get the USSEP patch here
Complete overhaul of the Werewolf system designed to balance existing mechanics and add powerful new abilities.

Apothecary - An Alchemy Overhaul

Installation instructions:

  1. Main File - Apothecary - An Alchemy Overhaul
  2. Main File - Apothecary - Food and Drink Addon (When prompted, select the name that matches the file on Nexus)
  3. Miscellaneous File - Apothecary - Fishing Patch (When prompted, select the name that matches the file on Nexus)
  4. Miscellaneous File - Apothecary - Food and Drink Addon - Survival Mode Patch (When prompted, select the name that matches the file on Nexus)
  5. Miscellaneous File - Apothecary - Food and Drink Addon - USSEP Patch (When prompted, select the name that matches the file on Nexus)
  6. Miscellaneous File - Apothecary - Rare Curios Patch (When prompted, select the name that matches the file on Nexus)
  7. Miscellaneous File - Apothecary - Saints and Seducers Patch (When prompted, select the name that matches the file on Nexus)
Complete overhaul of the alchemy, food, and alcohol systems designed to balance existing options while adding powerful new effects.

Pilgrim - A Religion Overhaul

Installation instructions:

  • Main File - Pilgrim - A Religion Overhaul
  • You will be prompted with a FOMOD installer, click Install
Complete overhaul of the religion system that adds dozens of new deities and with powerful shrine effects.

Thaumaturgy - An Enchanting Overhaul

Installation instructions:

  • Main File - Thaumaturgy - An Enchanting Overhaul
Complete overhaul of the enchanting system designed to balance existing enchantments and add powerful new weapon and armor enchantments

Hand to Hand - An Adamant Addon

Installation instructions:

  • Main File - Hand to Hand - An Adamant Addon
  • Optional Files - Hand to Hand - Armor Addon (When prompted, select the name that matches the file on Nexus)
  • Optional Files - Hand to Hand - Locksmith Addon (When prompted, select the name that matches the file on Nexus)
  • Optional Files - Hand to Hand - Trainers and Skill Books Addon (When prompted, select the name that matches the file on Nexus)
Addon for Adamant - A Perk Overhaul that merges Lockpicking and Pickpocketing into a single skill and adds a new perk tree for Hand to Hand combat.

Sensible Bribes

Installation instructions:

  • Main File - Sensible Bribes
Calculation of the bribe amount switched from using your level to your speechcraft.

Dynamic Mercenary Fees

Installation instructions:

  • Main File - Dynamic Mercenary Fees
Makes mercenaries ask for more or less money for their services depending on their level, your and their race and how good you are at bartering.
