Gameplay

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Gameplay.
It is especially important to read the descriptions of these mods (you should still read the description of every mod), so you know what has changed and how to use the new features.

Aetherius - A Race Overhaul

Installation instructions:

  • Main Files - Aetherius - A Race Overhaul
Complete overhaul of human, elf, and beast races designed to balance the races and add powerful new gameplay options.

Mysticism - A Magic Overhaul

Installation instructions:

  • Main Files - Mysticism - A Magic Overhaul
  • Optional Files - Mysticism - Jump Spell Addon (select the name that matches the file on Nexus when the installation prompt comes up)
  • Optional Files - Mysticism - Survival Spells Addon (select the name that matches the file on Nexus when the installation prompt comes up)
Complete overhaul of the magic system, designed to balance existing spells and add powerful new magical effects.

Adamant - A Perk Overhaul

Installation instructions:

  • Main Files - Adamant - A Perk Overhaul
  • Optional Files - Adamant - Smithing Addon (select the name that matches the file on Nexus when the installation prompt comes up)
Complete overhaul of the perk trees designed to balance existing skills and add powerful new perks.

Mundus - A Standing Stone Overhaul

Installation instructions:

  • Main Files - Mundus - A Standing Stone Overhaul
  • Optional Files - Mundus - USSEP Tower Stone Patch (select the name that matches the file on Nexus when the installation prompt comes up)
Complete overhaul of the Standing Stones designed to balance the stone bonuses and add powerful new gameplay options.

Arena - An Encounter Zone Overhaul

Installation instructions:

  • Main Files - Arena - An Encounter Zone Overhaul
Complete overhaul of the encounter zones designed to increase difficulty and provide an organic sense of progression.

Blade and Blunt - A Combat Overhaul

Installation instructions:

  • Main Files - Blade and Blunt - A Combat Overhaul
Complete overhaul of the combat designed to increase difficulty and encourage active resource management.

Scion - A Vampire Overhaul

Installation instructions:

  • Main Files - Scion - A Vampire Overhaul
  • Also get the USSEP Patch here
Complete overhaul of the Vampire system designed to balance existing mechanics and add powerful new abilities.

Manbeast - A Werewolf Overhaul

Installation instructions:

  • Main Files - Manbeast - A Werewolf Overhaul
  • Also get the USSEP Patch here
Complete overhaul of the Werewolf system designed to balance existing mechanics and add powerful new abilities.

Apothecary - An Alchemy Overhaul

Installation instructions:

  1. Main Files - Apothecary - An Alchemy Overhaul
  2. Main Files - Apothecary - Food and Drink Addon (select the name that matches the file on Nexus when the installation prompt comes up)
  3. Miscellaneous Files - Apothecary - Fishing Patch (select the name that matches the file on Nexus when the installation prompt comes up)
  4. Miscellaneous Files - Apothecary - Food and Drink Addon - Survival Mode Patch (select the name that matches the file on Nexus when the installation prompt comes up)
  5. Miscellaneous Files - Apothecary - Food and Drink Addon - USSEP Patch (select the name that matches the file on Nexus when the installation prompt comes up)
  6. Miscellaneous Files - Apothecary - Rare Curios Patch (select the name that matches the file on Nexus when the installation prompt comes up)
  7. Miscellaneous Files - Apothecary - Saints and Seducers Patch (select the name that matches the file on Nexus when the installation prompt comes up)
Complete overhaul of the alchemy, food, and alcohol systems designed to balance existing options while adding powerful new effects.

Pilgrim - A Religion Overhaul

Installation instructions:

  1. Main Files - Pilgrim - A Religion Overhaul
  2. You will be prompted with a FOMOD installer, click Install
Complete overhaul of the religion system that adds dozens of new deities and with powerful shrine effects.

Thaumaturgy - An Enchanting Overhaul

Installation instructions:

  • Main Files - Thaumaturgy - An Enchanting Overhaul
Complete overhaul of the enchanting system designed to balance existing enchantments and add powerful new weapon and armor enchantments.

Hand to Hand - An Adamant Addon

Installation instructions:

  • Main Files - Hand to Hand - An Adamant Addon
  • Optional Files - Hand to Hand - Locksmith Addon (select the name that matches the file on Nexus when the installation prompt comes up)
  • Optional Files - Hand to Hand - Trainers and Skill Books Addon (select the name that matches the file on Nexus when the installation prompt comes up)
Addon for Adamant - A Perk Overhaul that merges Lockpicking and Pickpocketing into a single skill and adds a new perk tree for Hand to Hand combat.
The Locksmith Addon will make a change to the Remember Lockpick Angle mod so it now requires a perk before the game will remember lockpick position.

Sensible Bribes

Installation instructions:

  • Main Files - Sensible Bribes
Calculation of the bribe amount switched from using your level to your speechcraft.

Dynamic Mercenary Fees

Installation instructions:

  • Main Files - Dynamic Mercenary Fees
Makes mercenaries ask for more or less money for their services depending on their level, your and their race and how good you are at bartering.

Vestige - An Artifact Overhaul

Installation instructions:

  • Main Files - Vestige - An Artifact Overhaul
  • You will be prompted with a FOMOD installer, click Next and then Install
Comprehensive overhaul and rebalance of Skyrim's Artifacts and Guild Items.
