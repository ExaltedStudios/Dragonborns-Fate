Initial Setup

In this step, we will do a clean installation and configuration of the game, which is required for the guide.

If you run into any issues at any point in the guide, please be sure to check the FAQ. If your issue is not addressed there, join the Discord for support.

Uninstalling the Game
If you do not have the game installed yet, skip this step.
  1. Open Steam and go to your Library
  2. Find The Elder Scrolls V: Skyrim Special Edition in the list
  3. Right-click on it and select Manage -> Uninstall
  4. Navigate to Steam\steamapps\common\ and, if present, delete the Skyrim Special Edition folder
  5. Navigate to Documents\My Games\Skyrim Special Edition and delete all the files ending in .INI

Installing the Game
  1. Open Steam and go to your Library
  2. Find The Elder Scrolls V: Skyrim Special Edition in the list and select Install
  3. Select Next then wait for the install to finish
REMINDER: Only legal copies of the Steam version of the game are supported! Bethesda.net, Xbox Game Pass, and pirated versions will not work with SKSE.

For full AE owners

You need to follow these steps only if you bought the extra Creation Club content

Installation instructions:

  1. Launch the game normally from steam and get to the main menu
  2. You will see a prompt asking you to download all the Creation Club content you bought
  3. Once everything is downloaded, close the game and follow the guide normally, making sure you pay attention to the "For full AE owners" section later on

Post-Installation

Downgrading to 1.5.97

In order to ensure compatibility with the widest amount of mods (Atleast for now) you will need to downgrade your game version, this also removes the need to worry about updates:


  • Download the Best of Both Worlds patcher from here
  • Run the executable file you just downloaded, then click Start Patching and close it when it's done
Make sure to download the right file! You only need Best of Both Worlds.

Key Terminology

Now that the game is installed, there are two folders from it that will be referred to in the guide often:


  • Root folder (where the game is installed): Steam\steamapps\common\Skyrim Special Edition
  • Data folder (where all of the game's assets are located): Steam\steamapps\common\Skyrim Special Edition\Data

Enabling File Extensions

By default, Windows Explorer will not show file extensions (such as .exe, .dll, or .esp). These extensions are very important when going through the guide, so it is highly recommended to enable file extensions:


  1. Open Windows Explorer
  2. Select the View tab at the top
  3. Check the box next to File name extensions

Generating Fresh .INI Files
  1. Run the game from Steam
  2. Click OK to both pop-ups that say Detecting Video Hardware then Exit

Fixing DPI scaling
  1. Right click on SkyrimSE.exe in the root folder.
  2. Open properties and navigate to the Compatibility tab.
  3. Click on Change high DPI settings.
  4. Set High DPI scaling override to Application.
