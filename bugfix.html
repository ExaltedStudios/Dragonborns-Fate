Bug Fixes

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Bug Fixes.

Unofficial Skyrim Special Edition Patch

Installation instructions:

  • Main File - Unofficial Skyrim Special Edition Patch
Fixes thousands of bugs with both the base game and DLCs.

If you get a missing masters warning in MO2 after installing this mod, then you used the wrong downgrader version.

Engine Fixes

Installation instructions:

  1. Download and install the Main File - (Part 1) SSE Engine Fixes for 1.5.39 - 1.5.97 normally through MO2
  2. Download the Main File - (Part 2) Engine Fixes - skse64 Preloader and TBB Lib 2020.3 file using the Manual Download button
  3. From the downloaded archive, extract all the files to the game's Root folder
  4. Once the files are installed, right-click on the SSE Engine Fixes mod in the left pane of MO2 and select Open in Explorer
  5. Navigate to SKSE\Pluginsand open EngineFixes.toml in a text editor (Such as notepad)
  6. Set CleanSKSECosaves to true (Line 5)
  7. Set MaxStdio to 8192 (Line 11)
  8. Set RegularQuicksaves to true (Line 13)
  9. Set SleepWaitTime to true (Line 18)
  10. Set SaveGameMaxSize to true (Line 66)
Fixes numerous bugs within the game's engine.
