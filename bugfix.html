Engine Fixes

Installation instructions:

  1. Download and install the Main File - (Part 1) SSE Engine Fixes for 1.5.39 - 1.5.97 5.8.3 normally through MO2
  2. Download the Main File - (Part 2) Engine Fixes - skse64 Preloader and TBB Lib 2020.3 file using the Manual Download button
  3. From the downloaded archive, extract all the files to the game's Root folder
  4. Once the files are installed, right-click on the SSE Engine Fixes mod in the left pane of MO2 and select Open in Explorer
  5. Navigate to SKSE\Plugins and open EngineFixes.toml in a text editor (such as notepad)
  6. Set CleanSKSECosaves to true (line 5)
  7. Set MaxStdio to 8192 (line 11)
  8. Set RegularQuicksaves to true (line 13)
  9. Set SleepWaitTime to true (line 18)
  10. Set SaveGameMaxSize to true (line 66)

- Fixes numerous bugs within the game's engine

Scrambled Bugs

Installation instructions:

  1. Download and install the Main File - Scrambled Bugs (Special Edition) normally
  2. Once the file is installed, right-click on it in the left pane of MO2 and select Open in Explorer
  3. Navigate to SKSE\Plugins and open ScrambledBugs.json in a text editor (such as notepad)
  4. Set "equipBestAmmo": to true (line 27)
  5. Set "steepSlopes": to true (line 42)
  6. Set "teammateDifficulty": to true (line 43)
  7. Set "underfilledSoulGems": to true (line 44)
  8. Save and exit

- Fixes many common bugs at an engine level, all fully configurable
