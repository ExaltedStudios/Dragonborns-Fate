Bug Fixes and Quality of Life Improvements

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator Bug Fix + QOL

Unofficial Skyrim Special Editon Patch

Installation instructions:

  • Main File - Unofficial Skyrim Special Edition Patch 4.2.4b

- Fixes thousands of bugs with both the base game and DLCs

Enchantment Equip Fix

Installation instructions:

  • Main File - Equip Enchantment Fix 1.2.2

- Fixes enchaments not properly applying when weapons/armors are equiped automatically

Enchanting Adjustments and Price-Charge Bug Fix

Installation instructions:

  1. Download the Enchanting Adjustments FOMOD V2.4 Installer file and begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, in which you should select only Price/Charge Bug Fix Only from the first page

- Fixes enchaments being worth more and using more charges after reloading the game

1st Person Sneak Strafe Stutter Fix

Installation instructions:

  • Main File - First Person Sneak Strafe-Walk Stutter Fix 1.0

- Fixes a bug that would cause the camera to stutter while diagonally strafing while crouched in first person

Quests Are In Skyrim

Installation instructions:

  • Main File - QuestsAreInSkyrim_USSEP_1_2_reupload 1.2

- Fixes base-game radiant quests being generated in DLC areas

Dragonborn Delayed
