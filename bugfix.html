Bug Fixes and Quality of Life Improvements

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator Bug Fix + QOL

Unofficial Skyrim Special Edition Patch

Installation instructions:

  • Main File - Unofficial Skyrim Special Edition Patch 4.2.4b

- Fixes thousands of bugs with both the base game and DLCs

Scrambled Bugs

Installation instructions:

  1. Download and install both the Main File - Scrambled Bugs 8 and Main File - Scrambled Eggchantments 1
If you do not know what to do when installing multiple mods from the same page, read the Mod Installation Advice section from the MO2 page
  1. Once the files are installed, right-click on the Scrambled Bugs mod in the left pane of MO2 and select Open in Explorer
  2. Navigate to NetScriptFramework\Plugins\ and open ScrambledBugs.json in a text editor (such as notepad)
  3. Set "equipBestAmmo": to true (line 14)
  4. Set underfilledSoulGems to true (line 17)
  5. Save and exit

- Fixes many common bugs at an engine level, all fully configurable
