Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator Bug Fixes

Unofficial Skyrim Special Edition Patch

Installation instructions:

  1. Once you click the link, the download will start automatically
  2. Once the file has finished downloading, install it normally through MO2
If you do not know how to install mods that do not have a Mod Manager Download button, read the Mod Installation Advice section from the MO2 page

- Fixes thousands of bugs with both the base game and DLCs

Scrambled Bugs

Installation instructions:

  1. Download and install the Main File - Scrambled Bugs (Special Edition) 17normally
  2. Once the file is installed, right-click on it in the left pane of MO2 and select Open in Explorer
  3. In the DLLPlugins folder, open ScrambledBugs.json in a text editor
  4. Set "underfilledSoulGems": to true (line 43)
  5. Save and exit

- Fixes many common bugs at an engine level, all fully configurable
