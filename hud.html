User Interface

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator HUD + UI

SkyUI

Installation instructions:

  • Main File - SkyUI_5_2_SE

- Modernizes and improves the user interface

SkyUI Flashing Savegame Fix

Installation instructions:

  • Main File - SkyUI SE - Flashing Savegames Fix 1.0

- Fixes the flashing savegame entries when trying to save/load a game with SkyUI

MoreHUD

Installation instructions:

  • Main File - moreHUD SE Light Master Version 4.0.0.5Beta

- Adds many new configurable elements to the HUD to show more useful info

MoreHUD Inventory Edition

Installation instructions:

  • Main File - moreHUD Inventory Edition Version 1.0.18

- Extends some features of moreHUD to the inventory menu

Immersive HUD

Installation instructions:

  • Main File - Immersive HUD - iHUD 0.2b

- Automatically hides/shows certain aspects of the HUD when needed

The default key for enabling the compass is X, but this can be changed in the Mod Configuration Menu

Quick Loot RE

Installation instructions:

  • Main File - QuickLootRE 2.8.6

- Adds a Fallout 4-style quick loot system

Better Dialogue Controls

Installation instructions:

  • Main File - Better Dialogue Controls v1_2

- Fixes annoyances with how dialogue controls

Better MessageBox Controls

Installation instructions:

  • Main File - Better MessageBox Controls v1_2

- Allows for better control over message boxes

Stay at System Page

Installation instructions:

  • Main File - Stay At System Page 1.5.0

- Changes the pause menu to default to the system page rather than the quests page

This mod does not completely work with controllers. If you plan on using a controller, it is best to skip this mod.

Whose Quest Is It Anyway

Installation instructions:

  • Main File - WhoseQuestIsItAnyway 1.3.0

- Specifies what quest an item belongs to if you try to drop it
