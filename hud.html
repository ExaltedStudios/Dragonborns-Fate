User Interface

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator UI.

SkyUI

Installation instructions:

  • Main Files - SkyUI_5_2_SE
  • Main Files - SkyUI SE - Flashing Savegames Fix
  • Main Files - FixNotesForSkyUI Skyrim AE
  • Main Files - SkyUI - Ghost Item Bug Fix
  • Main Files - Opt. 3 - Larger MCM Menu for SkyUI
Modernizes and improves the user interface.

MCM Helper

Installation instructions:

  • Main Files - MCM Helper SE
Requirement for many mods, also simplifies the creation of Mod Configuration Menus and extends them with powerful new features.

MoreHUD

Installation instructions:

  1. Main Files - moreHUD SE Light Master - AE
  2. Also get the Settings Loader here
  3. You will be prompted with a FOMOD installer, click Install
Adds many new configurable elements to the HUD to show more useful info.

MoreHUD Inventory Edition

Installation instructions:

  1. Main Files - moreHUD Inventory Edition - AE
Brings some features from moreHUD into the inventory menu.

Quick Loot RE

Installation instructions:

  • Main Files - QuickLootRE
Adds a Fallout 4-style quick loot system.

Better Dialogue Controls

Installation instructions:

  • Main Files - Better Dialogue Controls
Fixes annoyances with how dialogue controls.

Better MessageBox Controls

Installation instructions:

  • Main Files - Better MessageBox Controls
Allows for better control over message boxes.

Stay At The System Page NG

Installation instructions:

  • Main Files - Stay At The System Page
Changes the pause menu to default to the system page rather than the quests page.
This mod does not completely work with controllers. If you plan on using a controller, it is best to skip it.

Horse Stamina HUD

Installation instructions:

  • Main Files - Horse Stamina HUD SE
Displays your horse's current stamina when riding.

Better Third Person Selection

Installation instructions:

  • Main Files - Better Third Person Selection (AE - SE)
Overhauls 3rd person object selection.
