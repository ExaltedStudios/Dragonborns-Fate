SkyUI

- This is a collection of multiple mods in order to have SkyUI working perfectly

Installation instructions:

  • Main File - SkyUI_5_2_SE
  • Main File - SkyUI SE - Flashing Savegames Fix
  • Main File - FixNotesForSkyUI Skyrim SE 1.5.97
  • Main File - SkyUI - Ghost Item Bug Fix
  • Main File - Opt. 3 - Larger MCM Menu for SkyUI

- Modernizes and improves the user interface

MCM Helper

Installation instructions:

  • Main File - MCM Helper SE (1.5)

- Simplifies the creation of Mod Configuration Menus and extends them with powerful new features, such as persistent INI settings and hotkey registration

MoreHUD

Installation instructions:

  • Optional File - moreHUD SE Light Master- Pre AE
  • Also get the Settings Loader here (PapyrusUtil version)

- Adds many new configurable elements to the HUD to show more useful info

Quick Loot RE

Installation instructions:

  • Old File - QuickLootRE

- Adds a Fallout 4-style quick loot system

Better Dialogue Controls

Installation instructions:

  • Main File - Better Dialogue Controls

- Fixes annoyances with how dialogue controls

Better MessageBox Controls

Installation instructions:

  • Main File - Better MessageBox Controls

- Allows for better control over message boxes

Stay At The System Page

Installation instructions:

  • Main File - Stay At System Page

- Changes the pause menu to default to the system page rather than the quests page

This mod does not completely work with controllers. If you plan on using a controller, it is best to skip this mod.

Whose Quest Is It Anyway

Installation instructions:

  • Old File - Whose Quest Is It Anyway

- Specifies what quest an item belongs to if you try to drop it

Horse Stamina HUD

Installation instructions:

  • Main File - Horse Stamina HUD

- Displays your horse's current stamina when riding

Better Third Person Selection
