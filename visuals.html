Visuals

There may be less mods than you may be used to with a modded game, but there is good reason for that. Skyrim Special Edition already has beautiful visuals in my opinion. Any visual improvements you make to the game also come at a cost. ENBs may make the game look slightly better, but they can also tank performance depending on the preset. In a game that already has dynamic shadows, ambient occlusion, godrays, and more features that ENB add, I do not see them as necessary. Texture mods may improve some textures, but they can also cause performance drops and stuttering. They also create a great disparity between the new textures and the vanilla textures, both in quality and in style, making for some jarring texture transitions or seams. Overall, I do not think these types of mods are worth it.

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select All Mods -> Create Separator
  2. Name the separator Visuals
IMPORTANT! You can only use one of the following two sets of mods. ELFX + ELFX Fixes (the first one) looks much better in my opinion, but can have a performance impact on lower-end PCs. ELE (the second one) does not look as good, but should have zero performance impact even on weaker PCs. Choose whichever one better fits your system. For reference, I use ELFX with virtually no noticeable performance impact on my system with a Ryzen 7 1700, GTX 1060 6GB, and 16GB of RAM.

Enhanced Lights and FX

Installation instructions:

  1. Download the Main File - Enhanced Lights and FX 3.06 through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should only select ELFX Enhancer on the first page under the Main - Ambience section
  3. Finish the installation normally

- Greatly improves interior lighting/light sources

ELFX Fixes

Installation instructions:

  1. Download the Main File - ELFX Fixes v5.6 through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should just select Install
  3. Finish the installation normally

- Bug fixes and improvements for ELFX

REMINDER: You can use either the previous two mods OR the next one. You cannot use them together.

Enhanced Lighting for ENB

Installation instructions:

  • Main File - ELE - SE v1.2.2

- Makes interiors darker for a more immersive atmosphere at no performance cost

Although this mod was made for use with ENBs, it still works and looks fine without them.

Dimmer Fog

Installation instructions:

  1. Download the Main File - DimmerFog 2.1 through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should just select Install
  3. Finish the installation normally

- Heavily reduces the ambient fog effects to increase visibility and make interiors look more natural

Blended Roads

Installation instructions:

  1. Download the Main File - Blended Roads 1.7 through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should just select REALly Blended Roads
    • If you have a weaker PC, you may want to use the regaulr Blended Roads option to avoid any performance decrease
  3. Finish the installation normally

- Better blends to road into the landscape textures to avoid seams and gives them 3D meshes to look more natural

Realistic Water Two

Installation instructions:

  1. Download the Main File - Realistic Water Two SE v3.1.0 through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should just select Next then Install
  3. Finish the installation normally

- Creates visual distinctions between different bodies of water

Better Water for Realistic Water Two

Installation instructions:

  • Main File - Better Water - BC7 1.5

- Adds much more realistic water textures to RW2 without any performance impact

Wander - A Weather Mod

Installation instructions:

  1. Download the Main File - Wander - A Weather Mod 3.1 through MO2 and begin the installation normally
  2. You will be prompted with a FOMOD installer, in which you should select the following:
    • On the first page, select Next
    • On the second page, select Wander-lite
    • On the last page, just select Install
  3. Finish the installation normally

- Simple and lightweight weather mod that enhances and adds variety to the vanilla weather without straying too far from the vanilla feel

As visual mods are very subjective, not everyone will like Wander. If you want an alternative, I suggest checking out Cathedral Weathers and Seasons. It's a great mod and its fully compatible with the guide if you install it instead of Wander. Only one of them can be used at a time.
