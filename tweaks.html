Tweaks

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator.
  2. Name the separator Tweaks.

Quests Are In Skyrim

Installation instructions:

  • Main File - QuestsAreInSkyrim_USSEP_1_3

Fixes base-game radiant quests being generated in DLC areas.

Timing is Everything

Installation instructions:

  • Main File - Timing is Everything SE
  • Also get the Settings Loader here

Allows you to customize the starting conditions for various quests.

Realistic Conversations

Installation instructions:

  • Main File - Realistic Conversations

Prevents NPCs from constantly barraging you with greetings.

Quest NPCs Run

Installation instructions:

  • Main File - Quest NPCs Run

Makes slow walking quest NPCs you follow run.

Bounty Preview

Installation instructions:

  • Main File - Bounty Preview

Allows you to preview bounty missions from innkeepers and either accept/reject them.

All Thieves Guild Jobs Concurrently

Installation instructions:

  • Main File - All Thieves Guild Jobs Concurrently - USSEP Version
  • Also get the USSEP patch here (Install manually in MO2)

Allows you to accept all seven types of Thieves' Guild missions at once, greatly reducing the amount of time it takes to complete them all.

Simple Offence Suppression

Installation instructions:

  • Main File - Simple Offence Suppression SE

Prevents friendly NPCs from turning hostile on you if you accidentally hit them during combat.

Encounter Zones Unlocked

Installation instructions:

  • Main File - Encounter Zones Unlocked SE

Prevents the game from locking an area's encounter zone level when you first enter it.
