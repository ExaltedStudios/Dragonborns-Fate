Utilities

Make sure the A Dragonborn's Fate profile is active before proceeding.

Skyrim Script Extender

Installation instructions:

  1. Click the download link next to Current SE build 2.0.19 (runtime 1.5.97)
Make sure you are downloaded the SE version, not the Classic or VR versions.
  1. From the downloaded archive, extract all the files to the game's Root folder
If you do not know what the Root folder is, read the Key Terminology section from the Initial Setup page.

- Extends the scripting capabilities of the game

In order for the script extender to work, you need to launch the game using the skse64_loader.exe file. It will automatically be added to Mod Organizer 2 upon restarting it.

SKSE64 INI

Installation instructions:

  • Main File - No Warpaint tweak 1.2
If you do not know what these instructions mean, or how to install mods with MO2, read the Mod Installation Advice section from the MO2 page

- Enables a setting in SKSE that cleans your save of dirty OnUpdate() blocks from scripts

After installing a mod, make sure you check the box next to it in the left pane of MO2 to enable it

Address Libraries for SKSE Plugins

Installation instructions:

  • Main File - All in one 2

- Resource required for other SKSE plugins

SSE Engine Fixes

Installation instructions:

  1. Download and install the Main File - (Part 1) Engine Fixes 5.6.0 normally through MO2
  2. Download the Main File - (Part 2) Engine Fixes - skse64 Preloader and TBB Lib 2020.3 file using the Manual Download button
  3. From the downloaded archive, extract all the files to the game's Root folder

- Fixes numerous bugs within the game's engine

.NET Script Framework

Installation instructions:

  • Main File - NetScriptFramework SkyrimSE v14

- Engine framework required for some mods

SSE Display Tweaks

Installation instructions:

  • Main File - SSE Display Tweaks 0.4.15-1

- Allows for playing at framerates over 60, along with many other display-related tweaks

Scaleform Translation Plus Plus

Installation instructions:

  • Main File - ScaleformTranslationPP 1.4.1

- Resource for expanding SkyUI, a mod that will be installed later


REMINDER: You need to run the game through the SKSE option in Mod Organizer 2 from now on, NOT the Skyrim Special Edition option. You may need to restart MO2 for the option to appear.
