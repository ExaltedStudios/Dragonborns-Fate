Utilities

Make sure the A Dragonborn's Fate profile is active before proceeding.

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Utilities.

SKSE

Installation instructions:

  1. Old Files - Skyrim Script Extender (SKSE64) 2.0.20 (Manual Download)
  2. From the downloaded archive, extract the contents of the archive to the game's Root folder
Make sure that you are downloading version 2.0.20 from the "Old Files" section!
If you do not know what the Root folder is, read the Key Terminology section from the Initial Setup page.
You need to extract all of the files and folders contained in the archive.

Display Tweaks

Installation instructions:

  1. Main File - SSE Display Tweaks
  2. Once the mod has been installed, double-click on it in the left pane of MO2
  3. In the INI-Files tab, select SKSE/Plugins/SSEDisplayTweaks.ini
  4. Change the following options:
    • Set EnableTearing= to true (line 109)
    • Set FramerateLimit= to a value you want the game FPS to be locked to (line 204)
All-in-one solution for smooth FPS-unlocked gameplay, also includes many fixes and tweaks.
If you don't mind something more complex you can find a more in-depth methods to lock your FPS on this guide by Wall_SoGB.
