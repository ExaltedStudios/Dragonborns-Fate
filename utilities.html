  2. Name the separator Utilities

Skyrim Script Extender

Installation instructions:

  1. Click the download link next to Current Special Edition build 2.0.20 (game version 1.5.97):
  2. Extract the contents of the skse64_2_00_20 folder from the downloaded archive to the game's Root folder
Make sure you are extracting all of the files inside of the skse64_2_01_05 folder, not just the folder itself.
If you do not know what the Root folder is, read the Key Terminology section from the Initial Setup page.

- Extends the scripting capabilities of the game

In order for the script extender to work, you need to launch the game using the skse64_loader.exe file. It will automatically be added to Mod Organizer 2 upon restarting it.

Display Tweaks

Installation instructions:

  • Main File - SSE Display Tweaks
  • Navigate to SKSE\Plugins and open SSEDisplayTweaks.ini in a text editor (such as notepad)
  • Set EnableTearing to true (line 109)
  • Set FramerateLimit to a value you want the game FPS to be locked to (line 204)

- All-in-one solution for smooth FPS-unlocked gameplay, also includes many fixes

If you don't mind something more complex you can find a superior method to lock your FPS on this guide by Wall_SoGB.
